## Descripción
Portafolio personal construido con Next.js para mostrar proyectos, artículos y datos de contacto en un estilo tipo editor.

## Tecnologías
- Next.js
- React
- TypeScript
- CSS Modules

## Ejecutar en local
1. Instalar dependencias:
   - `npm install`
2. Levantar el entorno de desarrollo:
   - `npm run dev`
3. Abrir en el navegador:
   - http://localhost:3000

## Deploy
El proyecto se despliega automáticamente en Vercel. Para ver la versión en producción:
- https://portfolio-puce-seven-90.vercel.app/
